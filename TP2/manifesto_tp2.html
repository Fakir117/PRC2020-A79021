Resumo

Foi realizada uma ontologia acerca do Arquivo de Música Digital através de um ficheiro em XML. Para criar a ontologia foi usado o Protégé para criar as suas classes e as suas propriedades. Para popular a ontologia, foi utilizado XSLT que transformou o conteúdo do ficheiro XML em formato TTL. Para realizar as queries pretendidas, primeiro foi criado um repositório no GraphDB para posteriormente ser realizado as queries em SPARQL.
